## Table of Contents

## Android

### Font weight

When designing for Android, use **Skyscanner Relative Android**.

* Book
* Emphasized (for emphasised text)
* Heavy (for `font-size-xxl` only)

### Font size

<Tokens
  tokens={ANDROID_TOKENS}
  platform="android"
  matcher={({ name, category }) => category === 'typesettings' && name.includes('FONT_SIZE')}
/>
